#OSHack Takeaways

Akshay, Andre, John, Justin

The Problem

  • Deciding where to eat as a group is always a challenge
  • Difficulty in matching so many preferences
  • Existing apps give you the "top 10 best" places to eat but users still faced with the burden of choice

Our Hack

  • User tells us what they cannot eat
  • Query the yelp API for nearby good food
    • API is restrictive: no budget information
    • Scrape our own information
  • We decide for the user what the meal should be

What we learnt

  • How to interact with basic web microframeworks
  • How to pass information within a web app
  • How to scrape data from a page
  • How to interact with APIs

What next?

  • Self learning algorithm that recognizes when you like our recommendations
  • Allow multiple users in a group to access the web app
  • Implement persistence
